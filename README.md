# Alak

> Проактивная система управления состоянием для JavaScript/TypeScript

Alak — это библиотека управления состоянием с проактивностью, типизацией и dependency injection.

теп> [!IMPORTANT]
> Мастер версия **V6** находится в разработке. Готовы модули **quark** и **nucl**, модуль **atom** почти готов.
> Актуальная версия **v5** доступна в ветке **v5.1**.

## Философия V6: От Монолита к Частицам

Версия 6 представляет собой фундаментальный пересмотр архитектуры, вдохновленный физикой элементарных частиц:

1.  **Quark (Кварк)** — Фундаментальная частица состояния.
    *   Максимальная производительность (оптимизировано под V8).
    *   Минимальное потребление памяти.
    *   Работает как "кирпичик" для построения системы.

2.  **Nucleus (Ядро)** — Интеллектуальный контейнер.
    *   Обертка над Кварком, добавляющая "мозги".
    *   Использует систему **Kinds** (видов) вместо наследования классов. Вы собираете поведение из плагинов (например, `kind: 'stored history validated'`).
    *   Управляет вычислениями (**Fusion**) и побочными эффектами.

3.  **Quantum Bus** — Квантовая шина событий.
    *   Позволяет частям системы общаться реактивно, не зная друг о друге напрямую.
    *   Разделяет систему на **Realms** (Измерения), изолируя логику движка от UI.
