# Плагин `ComputedPlugin`

Этот плагин автоматически превращает все геттеры в классе модели в реактивные вычисляемые свойства (на базе `fusion`).

## Как это работает

Плагин использует систему отслеживания зависимостей (Dependency Tracking). При первом обращении к геттеру он фиксирует, какие другие свойства модели были прочитаны, и создает между ними связь.

```typescript
class Cart {
  price = 100;
  quantity = 2;

  // Станет вычисляемым нуклоном
  get total() {
    return this.price * this.quantity;
  }
}
```

## Преимущества
1.  **Кеширование**: Геттер не пересчитывается при каждом обращении. Результат хранится в кеше до тех пор, пока не изменятся зависимости.
2.  **Ленивость**: Если к геттеру никто не обращается, вычисление не производится.
3.  **Автоматизация**: Вам не нужно вручную перечислять зависимости в `fusion`.

## Особенности
- Геттер должен быть синхронным и зависеть только от других реактивных свойств этой же модели (или внешних нуклонов, прочитанных через `.value`).
- Для доступа к самому объекту вычисления используйте префикс `$`: `cart.$total` вернет инстанс нуклона.
