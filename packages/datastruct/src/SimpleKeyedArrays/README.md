# SimpleKeyedArrays

Простая и быстрая структура данных для хранения массивов по ключам. Оптимизирована для максимальной производительности.

## Особенности

- Максимальная производительность при итерации за счёт прямого доступа к методам массива
- Всегда возвращает массив, даже если ключ отсутствует (возвращает пустой массив)
- Поддерживает основные операции: push, get, remove, forEach, size, has
- Прямой доступ к массиву через `getArray` для максимальной производительности

## API

### `SimpleKeyedArrays()`

Создаёт новый экземпляр структуры данных.

### `push(key, value)`

Добавляет значение в конец массива по указанному ключу и возвращает индекс добавленного элемента.

### `get(key)`

Возвращает массив значений по ключу или пустой массив, если ключ отсутствует.

### `forEach(key, iterator)`

Вызывает переданную функцию для каждого элемента массива по ключу.

### `size(key)`

Возвращает длину массива по ключу.

### `has(key)`

Проверяет, существует ли массив по ключу.

### `remove(key, index)`

Удаляет элемент из массива по указанному индексу.

### `clearKey(key)`

Удаляет массив по ключу.

### `clearAll()`

Очищает всю структуру данных.

### `getArray(key)`

Возвращает внутренний массив по ключу для прямого доступа (максимальная производительность).
